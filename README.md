# Hypixel API

Kotlin Hypixel API wrapper

## Importing

Replace `VERSION` with the latest version above.

### Gradle

##### Groovy DSL

```gradle
repositories {
    jcenter()
}

dependencies {
    implementation 'com.github.mdashl:hypixel-api:VERSION'
}
```

##### Kotlin DSL

```kotlin
repositories {
    jcenter()
}

dependencies {
    implementation("com.github.mdashl:hypixel-api:VERSION")
}
```

### Maven

```xml
<depedencies>
    <dependency>
        <artifactId>hypixel-api</artifactId>
        <groupId>com.github.mdashl</groupId>
        <scope>compile</scope>
        <version>VERSION</version>
  </dependency>
</depedencies>

<repositories>
    <repository>
      <id>jcenter</id>
      <url>https://jcenter.bintray.com/</url>
    </repository>
</repositories>
```
